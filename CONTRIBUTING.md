# Contribuciones al repositorio

Este repositorio contiene el material docente del curso **Minería de Datos**.

El repositorio incluye:

- presentaciones de clase
- guías de trabajo
- datasets utilizados en el curso
- recursos complementarios

---

# Uso del repositorio

Los estudiantes pueden utilizar este repositorio para:

- descargar materiales de clase
- revisar las presentaciones utilizadas en las sesiones
- consultar documentación del proyecto del curso
- acceder a datasets utilizados en los ejercicios

Este repositorio funciona principalmente como **repositorio de apoyo al curso**, por lo que la mayor parte del contenido es publicado por el profesor.

---

# Reporte de errores

Si encuentras errores en:

- datasets
- material de clase
- documentos

puedes reportarlos utilizando la sección **Issues** del repositorio.
