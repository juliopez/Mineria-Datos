# Datasets del curso

Este directorio contiene los conjuntos de datos utilizados en las actividades prácticas del curso.

---

## Dataset: Deserción estudiantil

**Descripción:**  
Dataset con información académica y demográfica de estudiantes, utilizado para analizar factores asociados a la deserción en el primer año.

---

## Uso en el curso

- Actividad práctica Semana 2  
- Análisis exploratorio de datos (EDA)

---

## Consideraciones

- Dataset con fines educativos  
- Puede contener valores faltantes, duplicados o inconsistencias  
- Los datos pueden ser simulados o anonimizados  
- No representa información real de estudiantes  

---

## Variables (referenciales)

El dataset puede incluir variables como:

- Id
- Edad  
- Promedio de notas  
- Asistencia  
- Tipo igreso  
- Carrera
- Estado final  

---

## Propósito pedagógico

El dataset está diseñado para que el estudiante pueda:

- Identificar tipos de variables  
- Detectar problemas de calidad de datos  
- Explorar relaciones entre variables  
- Comprender el vínculo entre datos y decisiones
